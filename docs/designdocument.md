## 3. 게임 시스템

### 3.1. 게임 규칙

- 단어만 입력할 수 있다.
- 한방 단어는 입력할 수 없다.
- 한글만 입력할 수 있다.
- 최소 두 글자 이상이어야 한다.
- 게임에 참여한 한 명이 다음 단어를 제한 시간 안에 입력하지 못하면 승리한다.
- 하드 모드의 경우, 이미 사용한 단어를 한 번 더 사용하면 패배하게 된다. 



### 3.2. 랭크 시스템

게임에서 승리할 때마다 점수를 얻어 랭크를 올리는 방식이다. 단어를 입력하고 바로 다음 사람이 내가 입력한 단어를 받아치지 못하면 추가 점수를 얻는다.
